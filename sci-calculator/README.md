## Usage

Evaluates simple mathematical expressions.
The expression can consist of any of the following mathematical operators (spaces are ignored, evaluated in order):

    ()  parenthesis
    E   floating point
    ^   exponentiation
    *   multiplication
    /   division
    +   addition
    -   subtraction
